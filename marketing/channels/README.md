# Channels

This folder is for channel-specific launch notes and draft posts. Each channel
should have its own angle, not a pasted copy of the same announcement.

## Current Drafts

- `github.md` - GitHub release, pinned repo, and README-oriented copy.
- `hacker-news.md` - Show HN title and body drafts.
- `china.md` - Chinese launch copy and channel notes.

## Drafts To Add Later

- `reddit.md` - subreddit-specific drafts and rule checks.
- `product-hunt.md` - launch checklist and asset list.
- `blog.md` - founder-style story and product design rationale.

Before posting anywhere, check the current channel rules and adapt the post to
the community. Do not blast the same copy everywhere.

## Channel Fit

- GitHub: best first public surface because the release artifact, README, and
  source context already live there.
- Hacker News: useful after screenshots and download flow are polished; lead
  with a transparent builder note.
- Chinese channels: useful if Chinese screenshots and a practical usage story
  are ready.
- Product Hunt: wait until there is a proper landing page, gallery, and demo
  video.
