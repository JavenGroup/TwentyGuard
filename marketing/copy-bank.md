## Founder-Style Story Draft

I did not want another gentle reminder that I could dismiss without thinking.

The problem was not that I had never heard of the 20-20-20 rule. The problem was
the tiny negotiation that happened every time a reminder appeared: one more
compile, one more message, one more video, one more hour.

TwentyGuard is my attempt to make that negotiation harder without turning eye
care into a noisy productivity system. It sits quietly in the macOS menu bar,
then uses a full-screen break overlay when it is time to rest. You can postpone,
but not forever. The health report focuses on the behavior that matters: whether
breaks were completed, how often they were delayed, and whether any work stretch
became too long.

The app also includes an optional night screen lock. In the evening it can
gradually shorten allowed screen-use time, then fully block screen use until the
configured morning time.

TwentyGuard is not medical software, and it does not promise to cure eye strain.
It is a small native Mac utility for people who already know they need breaks and
want a tool firm enough to make those breaks happen.

## FAQ Draft

### Is TwentyGuard a medical app?

No. It is a general wellness and screen-rhythm utility. It does not diagnose,
treat, or cure any condition.

### Does it need network access?

No network access is needed for the core behavior. Settings, logs, and
statistics are stored locally on your Mac.

### Can I postpone breaks?

Yes. Breaks can be postponed by 1, 2, or 5 minutes, but postpones are capped so
they cannot be delayed forever. The default cumulative limit is 5 minutes.

### Does it work with multiple monitors?

Yes. The break overlay is designed to cover all connected displays.

### What is night screen lock?

It is an optional evening boundary. TwentyGuard can gradually shorten screen-use
time before a configured lock time, then fully block screen use until morning.

### Why direct download instead of App Store only?

The current release path is direct download. The v1.5.0 DMG is documented as
Developer ID signed, Apple-notarized, stapled, and accepted by Gatekeeper.
